¿Cómo funciona?
Sube uno o varios PDFs. El parser extrae todos los deals que encuentre en cada archivo (fechas, apellidos, stock, Comm#1, F&I, ajustes, hold reason). Los registros válidos se guardan automáticamente al dashboard. Los duplicados exactos se saltan y se marcan en la tabla. Los PDFs imagen-only muestran opción de captura manual.
Merge New Records from JSON
Paste a JSON array of new records. Composite keys are checked — exact duplicates are always skipped. Safe to run multiple times.
Scan & Remove Duplicates
Scans all records using the composite key (date_clientname_total). Exact matches are listed and can be purged.
Data Management
Export all data as JSON for backup, or reset to seed data.
